Server Name: PAPT1
Host: host.docker.internal
Port: 5432
Database: pap-lectores
User: lectoruy



La propiedad hbm2ddl.auto en hibernate.cfg.sml con el valor update es clave para que Hibernate cree automáticamente las tablas. 
Si esta propiedad no está configurada, las tablas no se generarán.







PGAdmin + Postgres:

Descargar pgadmin para windows: pgadmin4-9.7-x64.exe
Instalar

Eliminar ambos contenedores. 

Crear de nuevo el contenedor postgres asi (sin la red):
docker run --name pg_db -e POSTGRES_USER=lectoruy -e POSTGRES_PASSWORD=pap12345 -e POSTGRES_DB=pap-lectores -p 5432:5432 -d postgres
